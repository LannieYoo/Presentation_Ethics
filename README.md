# Ethical AI Impact Assessment - Presentation Website

13분 이내 발표용 웹사이트

## 사용 방법

1. **웹사이트 열기**: `index.html` 파일을 브라우저에서 엽니다.

2. **슬라이드 네비게이션**:
   - **화살표 버튼**: 하단의 Prev/Next 버튼 클릭
   - **키보드**:
     - `←` 또는 `↑`: 이전 슬라이드
     - `→` 또는 `↓`: 다음 슬라이드
     - `Home`: 첫 슬라이드로 이동
     - `End`: 마지막 슬라이드로 이동

3. **발표 시간 배분 (13분 기준)**:
   - **Member1 (슬라이드 1, 12)**: ~2분 (Intro 1분, Conclusion 1분)
   - **Member2 (슬라이드 3-5)**: ~4분 (시스템 배경 1분, 이해관계자 1.5분, 데이터 흐름 1.5분)
   - **Member3 (슬라이드 6-8)**: ~3.5분 (윤리적 위험 1.5분, 위험 등록부 1분, 규제 1분)
   - **Member4 (슬라이드 9-11)**: ~3.5분 (완화 전략 1.5분, 거버넌스 1분, 권고사항 1분)

## 파일 구조

- `index.html`: 메인 HTML 파일 (Tailwind CSS 사용)
- `script.js`: 슬라이드 네비게이션 스크립트
- `docs/Report.md`: 원본 보고서 (참고용)

## 주요 기능

- 슬라이드 형식의 발표 웹사이트
- 키보드 및 버튼 네비게이션
- 반응형 디자인 (모바일/태블릿 지원)
- 멤버별 섹션 구분 (배지로 표시)
- 시각적 강조 (위험 카드, 표, 플로우차트)

## 브라우저 호환성

- Chrome, Firefox, Safari, Edge (최신 버전)
- 모바일 브라우저 지원
